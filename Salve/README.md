# Site do Psicólogo Rafael Marrero Brignol

## Requisitos
- Node.js 18+ e npm.

## Scripts
- `npm install` — instala dependências de desenvolvimento.
- `npm run dev` — sobe servidor de desenvolvimento servindo a raiz do projeto.
- `npm run build` — gera versão minificada em `dist/` (HTML copiado, CSS minificado, JS minificado).
- `npm run dev:dist` — serve os arquivos já gerados em `dist/`.

## Estrutura
- `index.html` — página principal.
- `styles.css` — estilos separados.
- `main.js` — lógica simples para ano no rodapé.
- `dist/` — saída de build, criada pelo script.

Nenhuma dependência de runtime é necessária; tudo é front-end estático.
